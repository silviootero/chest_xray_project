
# Tarea 2: Procesamiento de Imágenes Digitales

##  Descripción del Proyecto
Este proyecto es una iniciativa de entrenamiento de un **modelo predictor de neumonía** a partir de un conjunto de **radiografías clasificadas**. El objetivo principal es desarrollar y evaluar un modelo de **aprendizaje automático (Machine Learning)** capaz de clasificar las radiografías de tórax para identificar la presencia de neumonía.

##  Componentes Clave
* **Adquisición y Preprocesamiento de Imágenes:** Carga y preparación del dataset de radiografías.
* **Diseño del Modelo:** Implementación y arquitectura de los modelos requeridos.
* **Entrenamiento y Evaluación:** Proceso de entrenamiento de los modelos y análisis de sus métricas de rendimiento.
